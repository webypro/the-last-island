The Last Island

A slow, atmospheric, choice-based 2D story adventure. Explore the island, read memories, make choices, solve light narrative puzzles, and reach different endings depending on what you do.

This is a browser version meant for GitHub Pages. It uses no external assets and saves locally in the browser.

Core features

Virtual joystick, smooth camera, notebook, dialogue choices, autosave, chapter progression, and multiple endings.

How to play

Move with the joystick. Tap Interact near NPCs or story objects. Open the Notebook to read memories, symbols, and choices.

Story structure

Arrival → Old Man → Child → Sailor → Island Reacts → Final Choice → Ending.
