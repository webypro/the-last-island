# The Last Island (Web Prototype)

This folder contains a single-file browser game prototype for **The Last Island**.

## Files
- `index.html` – the playable game
- `README.md` – this guide

## How to use on GitHub Pages
1. Create a GitHub repository.
2. Upload `index.html` to the repository root.
3. Enable GitHub Pages from the repository settings.
4. Open the published URL in your browser.

## What is included
- Main menu
- New Game / Continue
- Virtual joystick
- Smooth camera follow
- Dialogue and choice system
- Notebook / memory system
- Autosave using browser localStorage
- Chapter progression
- Multiple endings
- No external assets required

## Controls
- Drag the joystick to move
- Tap **Interact** to talk or choose
- Tap **Notebook** to open memories
- Keyboard also works on desktop:
  - WASD / Arrow keys = move
  - E / Enter / Space = interact
  - N = notebook

## Note
This is a browser prototype meant to be uploaded directly to GitHub Pages. It does not need any extra assets to run.
